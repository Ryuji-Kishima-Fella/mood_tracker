# mood_tracker

A simple Python app to record and track my daily moods.


## Features
- Log how you feel each day
- View your modd history
- Saves data locally in 'mood_log.txt'

## Future Improvements
- Add charts/graphs
- Add daily reminders
- Turn into a web or mobile app

## How to Run
1. Install Python 3
2. Run 'py mood_tracker.py' in your terminal.
