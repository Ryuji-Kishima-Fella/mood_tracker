# mood_tracker

A simple Python program that helps you log and track your daily moods - made as my first GitHub project.

## About the Project
This console-based app lets you:
- Record how how you feel each day
- View your mood history
- NEW!: See a summary of how often each mood occurs
- Identify your most frequent mood 
- Saves data locally in 'mood_log.txt'

## Future Improvements
- Add charts/graphs
- Add daily reminders
- Turn into a web or mobile app

## How to Run
1. Install Python 3.10+
2. Download or clone this repository:
```bash
git clone https://github.com/Ryuji-Kishima-Fella/mood-tracker.git
```
3. Run 'py mood_tracker.py' in your terminal.
